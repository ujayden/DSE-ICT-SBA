Learning Path

To maximize your effectiveness of learning, we provide a learning path for you to follow. The Dynamic Learning Path will help you to learn the course with the best order.

Everyone has their own learning path. The learning path will be generated based on your learning progress and your learning style.

For more information, please click the learn more button to see learning path specific for different course.

Here is a diagram to show the general structure of our learning path.

  1. Beginner

    Goal: Attain to basic performance

    • Learn the basic knowledge and concept
    • Relax and Interactive learning

  2. Basic

    Goal: Attain to average performance

    • Explain the knowledge and concept
    • Learn difficult concept

  3. Advanced

    • Explain the knowledge and concept
    • Learn abstract and difficult concept
    • Learn the exam skills

  4. Expert

    • Learn more exam skills
    • Revise the knowledge

Resource Library

Resource Library is a place for you to find resources for learning. You can find the resources that you need for learning. When you finish a level of difficulties, you can find the resources for the previous level of difficulties.

This allow you to review the knowledge that you have learnt before. Don't forget to check out the resource library for more resources, as additional supplement resource will be added to the resource library!
